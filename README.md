# Customer Segmentation & RFM Analysis

A data analytics project to segment customers using RFM (Recency, Frequency, Monetary) analysis and K-Means clustering — processing 100K+ transactions to identify high-value, loyal, and at-risk customers for targeted campaign strategies.

---

## Results Summary

| Segment | Customers | Avg Spend | Strategy |
|---|---|---|---|
| Champions | High frequency, recent buyers | Highest | Reward & retain |
| Loyal Customers | Regular buyers, high spend | High | Upsell & referral |
| Potential Loyalists | Moderate frequency | Medium | Frequency incentives |
| At-Risk Customers | Haven't bought recently | Low | Win-back campaigns |

## Dataset

**Online Retail II** — UCI Machine Learning Repository  
Source: [Kaggle — Online Retail II UCI](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)

- 1,000,000+ transactions
- 8 features: Invoice, StockCode, Description, Quantity, InvoiceDate, Price, CustomerID, Country
- Date range: 2009–2011

## How to Run

**1. Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/customer-segmentation-rfm.git
cd customer-segmentation-rfm
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Download the dataset**  
Go to [Kaggle](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci), download `online_retail_II.xlsx` and place it in the `data/` folder.

**4. Run the notebook**
```bash
python -m notebook notebooks/rfm_analysis.ipynb
```

Run all cells top to bottom. Two CSV files will be generated in the `data/` folder.

**5. Open Power BI**  
Import `rfm_segments.csv` and `monthly_revenue.csv` into Power BI Desktop to build dashboards.

## Key Findings

- **Champions** drive the highest revenue despite being a small % of customers
- **At-Risk customers** were once frequent buyers — win-back campaigns can recover 10–15% of them
- **AOV (Average Order Value)** varies significantly by country and segment
- Customers with recency < 30 days have 3x higher lifetime value than inactive ones

## Campaign Targeting Strategies

| Segment | Strategy |
|---|---|
| Champions | Reward program, early access to new products, brand ambassador |
| Loyal Customers | Upsell premium plans, referral discounts, loyalty points |
| Potential Loyalists | Onboarding offers, frequency-based incentives |
| At-Risk Customers | Win-back email campaigns, personalized discounts |
